# 十勝地方の河川水位可視化

2016年8月に北海道を直撃した台風による各河川の水位への影響を可視化しました。[Github pages](colspan.github.io/tokachi-rivers-water-level/)にて公開しています。

## 対象台風
 
 - 台風 5号 (2016/08/09 接近)
 - 台風 6号 (2016/08/15 接近)
 - 台風 7号 (2016/08/17 上陸)
 - 台風11号 (2016/08/21 上陸)
 - 台風 9号 (2016/08/23 上陸)
 - 台風10号 (2016/08/30 上陸)

## 対象期間

2016/08/01 01:00 ～ 2016/09/05 22:00
